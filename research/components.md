Set `auto_mode: true` to let ElevenLabs handle chunk buffering automatically. Audio comes back as base64-encoded chunks.\n\n```\nLLM tokens → SSE to browser → WebSocket to ElevenLabs → Audio chunks → AudioContext playback\n```\n\n**Interrupt/cancel pattern:**\nThe Multi-Context WebSocket API allows independent audio streams over a single connection. When the user starts speaking (detected by VAD), close the current context to stop audio immediately, then open a new context for the next response.\n\n**Audio playback with Web Audio API:**\n\n```tsx\n// Simplified streaming audio player\nclass StreamingAudioPlayer {\n  private context: AudioContext;\n  private queue: AudioBuffer[] = [];\n  private playing = false;\n\n  constructor() {\n    this.context = new AudioContext();\n  }\n\n  async enqueue(base64Audio: string) {\n    const binary = atob(base64Audio);\n    const buffer = new Uint8Array(binary.length);\n    for (let i = 0; i < binary.length; i++) {\n      buffer[i] = binary.charCodeAt(i);\n    }\n    const audioBuffer = await this.context.decodeAudioData(buffer.buffer);\n    this.queue.push(audioBuffer);\n    if (!this.playing) this.playNext();\n  }\n\n  private playNext() {\n    if (this.queue.length === 0) { this.playing = false; return; }\n    this.playing = true;\n    const buffer = this.queue.shift()!;\n    const source = this.context.createBufferSource();\n    source.buffer = buffer;\n    source.connect(this.context.destination);\n    source.onended = () => this.playNext();\n    source.start();\n  }\n\n  cancel() {\n    this.queue = [];\n    this.playing = false;\n    // AudioContext.close() and recreate for hard stop\n  }\n}\n```\n\n**Fallback: `SpeechSynthesis` API:**\nBrowser-native, free, offline. break; }\n\n      buffer += decoder.decode(value, { stream: true });\n      const lines = buffer.split(\"\\n\");\n      buffer = lines.pop() || \"\"; // Keep incomplete line in buffer\n\n      for (const line of lines) {\n        if (!line.startsWith(\"data: \")) continue;\n        const payload = line.slice(6);\n        if (payload === \"[DONE]\") { onDone();
